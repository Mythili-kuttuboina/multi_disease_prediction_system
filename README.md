# multi_disease_prediction_system
Multi Disease Prediction System using Machine Learning (Streamlit Web App)

📖 Project Description

The Multi Disease Prediction System is an interactive web-based application developed using machine learning techniques to predict the likelihood of multiple diseases such as diabetes, heart disease, and Parkinson’s disease. The frontend of the application is built using Streamlit, which provides a simple and intuitive interface for users to input medical data and receive real-time predictions.

The system integrates multiple trained machine learning models into a unified platform, allowing users to seamlessly switch between different disease prediction modules. By leveraging data preprocessing, model training, and real-time inference, this application demonstrates the practical implementation of machine learning in the healthcare domain.

🎯 Objectives
To build a web-based system for predicting multiple diseases
To provide a user-friendly interface using Streamlit
To enable early detection of diseases through ML models
To demonstrate real-world application of machine learning in healthcare
⚙️ Technologies Used
Programming Language: Python
Frontend: Streamlit
Libraries: NumPy, Pandas, Scikit-learn, Pickle
IDE: Visual Studio Code
Version Control: Git
Repository: GitHub
🧠 Machine Learning Models

The system includes separate prediction modules for:

Diabetes Prediction
Heart Disease Prediction
Parkinson’s Disease Prediction

Algorithms used:

Logistic Regression
Support Vector Machine (SVM)
Random Forest
💻 Key Features
Interactive UI using Streamlit
Multiple disease prediction in one platform
Instant results based on user input
Easy navigation using sidebar
Scalable and modular design

📊 How It Works
User selects a disease prediction option from the sidebar
Inputs required medical parameters
Data is processed and passed to the trained model
Model predicts the result
Output is displayed instantly on the screen
